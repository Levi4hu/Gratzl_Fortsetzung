* Gruppe: 3

* Projektthema: Nachbarschaftsmarktplatz für lokale Dienstleistungen

### Implementierung
Framework: Android  
API-Version: Android API-Level 26+ (36)
Geräte, auf dem getestet wurde: Pixel 6,9 (Emulator), physisches Android-Gerät

Externe Libraries und Frameworks:
- **Jetpack Compose (Material 3)** – UI Framework
- **Coil 2.6.0** (`io.coil-kt:coil-compose`) – Asynchrones Laden von Netzwerkbildern
- **Nunito** (Google Fonts) – Schriftart, SIL Open Font License (OFL)
- **Unsplash** – Platzhalterbilder in den Listing Cards, Unsplash License (kostenlose kommerzielle Nutzung)
- **AndroidX ViewModel & StateFlow** – MVVM Architektur

Dauer der Entwicklung: ca. 260 Stunden

Weitere Anmerkungen:
- Architektur: MVVM (Model-View-ViewModel) mit Jetpack Compose
- Die App wurde für Wien entwickelt und zeigt Nachbarschafts-Dienstleistungen (Angebote & Anfragen) nach Bezirk gefiltert
- Alle Daten sind Sample-Daten (kein Backend)
